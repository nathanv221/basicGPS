From this page you can download the following files for your use in our term project. It is probably most convenient to download the first file and unzip it. But you can get the files individually, too.
  1. A zip file containing all other files in this list
  2. README: do it!
  3. vehicle.class: executable of my vehicle
  4. receiver.class: executable of my own receiver
  5. satellite.class: executable of my own satellite
  6. angles.class: executable of my own angles class
  7. data.dat: the data file
  8. b12.dat: trip to B12 in one step
  9. b12t0.dat: trip to b12 in one step, at time 0
  10. bm.dat: trip to Black Mountain, in 90 steps
  11. np.dat: a trip to the north pole in 15 steps
  12. sp.dat: a trip to the South Pole in 15 steps
  13. o.dat: a trip to the point O in 9 steps

On a Unix system, with java installed, you can uses these programs by giving a command like: 
cat bm.dat | java vehicle | java satellite | java receiver
This command should produce output that is almost identical to the contents of the file vehicle.dat which is also produced by the above command.
 
If all goes well you will be able to replace one or both of java satellite and java receiver with suitable commands that execute your own satellite or receiver.
 
The file 'bm.dat' in the above command can be replaced by any of the data files listed above in items 7 through 12. You can also create your own data files. They have the following contents (all numbers in one line, separated by blank spaces).
  1. beginning time
  2. ending time
  3. number of steps of trip
  4. degrees latitude
  5. minutes latitude
  6. seconds latitude
  7. hemisphere (1 if Northern, -1 if Southern)
  8. degrees longitude
  9. minutes longitude
  10. seconds longitude
  11. hemisphere (1 if Eastern, -1 if Western)
  12. altitude of destination, in meters
This will generate a trip to the given position in the given number of steps, starting at the lamp post B12.
